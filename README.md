Task 2 :  Weather Information App (Python)

## 📌 Project Description

This project fetches real-time weather information using the OpenWeatherMap API. It demonstrates API integration, JSON parsing, and user input handling in Python.

---

## Features

- Fetch weather using Requests module
- Parse JSON response
- Search weather by city
- Display:
  - Temperature
  - Humidity
  - Wind Speed
  - Weather Condition
  - Country

---

## Technologies Used

- Python
- Requests Library
- OpenWeatherMap API

---

## Installation

Clone the repository

```bash
git clone https://github.com/YourUsername/Weather-App.git
```

Install dependency

```bash
pip install requests
```

Run

```bash
python weather.py
```
